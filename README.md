# 📈 Stock Trading Platform

## 📌 Project Description

The Stock Trading Platform is a Java-based console application developed to simulate a basic stock market trading environment. This project allows users to buy and sell stocks, manage their investment portfolio, and track available balance through a simple and interactive console interface.

The application demonstrates the implementation of Object-Oriented Programming (OOP) concepts, Java collections, loops, conditional statements, and user input handling. It is designed as a beginner-friendly project for understanding how stock trading systems and portfolio management applications work in real-world scenarios.

Users can view available stocks with their prices, purchase shares based on available balance, sell owned shares, and monitor their portfolio performance dynamically.

---

## 🚀 Features

- Buy stocks using available balance
- Sell owned stocks
- View portfolio details
- Track remaining account balance
- Simulate stock market environment
- Dynamic data management using ArrayList and HashMap
- Console-based user-friendly interface
- Object-Oriented Programming implementation
